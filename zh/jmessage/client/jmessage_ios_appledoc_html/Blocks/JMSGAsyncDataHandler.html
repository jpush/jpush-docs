JMSGAsyncDataHandler

数据返回的异步回调

typedef void (^JMSGAsyncDataHandler) (NSData *data, NSString *objectId, NSError *error)

Discussion

专用于数据返回 API.

  • objectId 当前数据的ID标识.
  • 消息里返回 voice/image 的 media data 时, objectId 是 msgId;
  • 用户里返回 avatar 头像数据时, objectId 是 username.
