JMSGNotificationEvent Class Reference

Inherits from NSObject

Overview

服务器端下发的事件通知, 如：用户被踢下线,加好友, SDK 作为一个通知事件下发. 上层通过 JMSGEventDelegate 类中的 onReceiveNotificationEvent: 代理方法接收事件,详见官方文档.

注意：消息事件(如：群事件)，SDK 依旧作为一个特殊的消息类型下发，依旧通过 JMSGMessageDelegate 接收消息事件.
