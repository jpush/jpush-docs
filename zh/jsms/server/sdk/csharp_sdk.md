# JSMS API CSharp Client

这是 [JSMS](https://docs.jiguang.cn/jsms/guideline/JSMS_guide/) Rest API 的 CSharp 版本封装开发包，由极光推送官方提供，支持最新的 API 功能。

## Installation
1. 添加 jsms.dll
2. 添加 [Json.NET](http://www.newtonsoft.com/json)

## Example
项目目录下的 example 文件夹中有示例代码。
